# Shukuma Exercise App

A comprehensive fitness tracking application built with React, Redux, and modern web technologies.

## Features

- **Workout Tracking** - Log and track your workouts with detailed analytics
- **Challenges** - Join community challenges and compete with friends
- **Fitness Journal** - Document your fitness journey with daily entries
- **Nutrition Tracking** - Monitor your meals
- **Progress Analytics** - Visualize your fitness progress over time
- **Social Features** - Connect with friends and share achievements

## Tech Stack

- **Frontend:** React 19, React Router, Redux Toolkit
- **Styling:** CSS
- **Charts:** Recharts
- **Icons:** Lucide React
- **Build Tool:** Vite
- **Code Quality:** ESLint, Prettier

## Getting Started

### Prerequisites

- Node.js 22+ and npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Fortune101-ai/shukuma-exercise-app.git
```
2. Navigate to the frontend directory
```bash
cd frontend/
```

3. Start the development server:
```bash
npm run dev
```

The app will then be running at `http://localhost:5173`

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
